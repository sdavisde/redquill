## Additional coverage beyond the minimum bullets

- `diff::model::tests::parse_patches_and_summarize_aggregate_counts` — 1.7
  (`parse_patches` + `summarize`, aggregate file/hunk/added/removed counts).
- `diff::model::tests::empty_diff_yields_empty_summary` — spec §6 edge case
  (empty diff / empty hunk lists).
- `diff::model::tests::crlf_in_content_preserved_verbatim` — spec §6 edge
  case (CRLF / embedded control chars preserved verbatim in `content`).
  This test caught a real bug during implementation: `str::lines()`
  normalizes `\r\n` and strips the `\r`, which would have silently
  corrupted CRLF file content. Fixed by splitting on the `\n` record
  separator only (stripping at most one trailing `\n`), never touching
  embedded `\r`.
